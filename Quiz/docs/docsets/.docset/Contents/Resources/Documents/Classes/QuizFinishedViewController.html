QuizFinishedViewController

class QuizFinishedViewController: PageViewContent

The last (right most) of the cards displayed by the pageViewController. It displays a simple prompt telling the user that he/she has reached the end of the quiz. An Exit button is displayed that will bring the user to the results screen
